# Truth Table Generator
 This is a JavaScript program which will generate a truth table given a well-formed formula of truth-functional logic.

## Symbols
|Symbol|Operation|Aliases|Example
|:-------:|:--------:|:---------:|:---------:|
|&&|AND|*, AND, ^|A && B
|\|\||OR|+, OR|A \|\| B
|=>|IMPLIES|->, IMPLIES|A => B
<=>|IF|<->, IF|A <=> B
!|NOT|~, NOT|!A
